# Fyers Historical Data Downloader

This project downloads historical Indian stock market data using the Fyers API and saves it locally as CSV files.

Files:
- fyers_downloader.py: main script to fetch historical data
- requirements.txt: Python dependencies

Usage:
1. Install dependencies: `pip install -r requirements.txt`
2. Set environment variables: FYERS_API_KEY, FYERS_API_SECRET, FYERS_ACCESS_TOKEN
3. Run: `python fyers_downloader.py`
